# Repo Template

## Suggested Files

- `README.md`
- `LICENSE`
- `CONTRIBUTING.md` where relevant
- `docs/` for deeper documentation
- `examples/` where interpretation or usage benefits from concrete references

## Suggested Review Flow

1. Apply the general checklist.
2. Apply the repo-specific checklist.
3. Verify metadata on the hosting platform.
4. Publish only when the baseline is clear and complete.
