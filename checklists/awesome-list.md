# Awesome List Checklist

## Overview

Use this checklist for curated lists and reference collections.

## Required

- [ ] Repository clearly states that it is a curated list.
- [ ] Scope is defined.
- [ ] Sections are logically organized.
- [ ] Each entry includes a concise description.
- [ ] Links are valid at time of review.
- [ ] License is appropriate for curated content.

## Recommended

- [ ] Contribution guidance explains what qualifies for inclusion.
- [ ] Formatting is consistent across sections.
- [ ] Intro explains who the list is for.
- [ ] Related resources or companion lists are linked.
- [ ] Optional criteria for inclusion are documented.

## Advanced

- [ ] Link checking workflow exists.
- [ ] Duplicate-link review process exists.
- [ ] Philosophy or editorial standard is documented.
