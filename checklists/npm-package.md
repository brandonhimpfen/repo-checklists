# NPM Package Checklist

## Overview

Use this checklist for JavaScript and Node.js packages published to npm.

## Required

- [ ] `package.json` includes name, version, description, and license.
- [ ] Repository, homepage, and bugs URLs are set where appropriate.
- [ ] README includes install instructions.
- [ ] README includes usage example.
- [ ] Exported API is documented or self-evident.
- [ ] Published files are intentionally scoped.

## Recommended

- [ ] Test script exists.
- [ ] Keywords are set.
- [ ] Type information or examples are included where useful.
- [ ] Versioning approach is clear.
- [ ] Basic release process is documented.

## Advanced

- [ ] Changelog is maintained.
- [ ] CI validates tests before release.
- [ ] Package includes multiple usage examples.
